# Packages

Third-party packages maintained by community members. No personal
responsibility is taken for them — they are kindly maintained by contributors.

## Distributions

### Arch Linux
[Arch Linux](https://archlinux.org/packages/extra/x86_64/keyd/) — maintained by Arch packagers.

### Debian
Available in Debian 13 ("trixie") and later:
```sh
sudo apt install keyd
```

### Ubuntu
Available in Ubuntu 25.04 ("plucky") and later:
```sh
sudo apt install keyd
```

For backports to other Ubuntu releases, use the
[`ppa:keyd-team/ppa`](https://launchpad.net/~keyd-team/+archive/ubuntu/ppa) archive.

### OpenSUSE
```sh
sudo zypper in keyd
```
[Download page](https://software.opensuse.org//download.html?project=hardware&package=keyd)
— maintained by [@bubbleguuum](https://github.com/bubbleguuum).

### Fedora
[COPR](https://copr.fedorainfracloud.org/coprs/alternateved/keyd/) — maintained by [@alternateved](https://github.com/alternateved).

### Alpine Linux
[keyd](https://pkgs.alpinelinux.org/packages?name=keyd) — maintained by [@jirutka](https://github.com/jirutka).

### Gentoo
[GURU](https://gitweb.gentoo.org/repo/proj/guru.git/tree/app-misc/keyd) — maintained by [jack@pngu.org](mailto:jack@pngu.org).

### Void Linux
```sh
sudo xbps-install -Su keyd
```
[xbps](https://github.com/void-linux/void-packages/tree/master/srcpkgs/keyd) — maintained by [@Barbaross](https://gitlab.com/Barbaross).

## Add Your Distribution

If you maintain a keyd package, please open a PR to add it to this list.
The canonical packaging status tracker is linked from the README:
<https://repology.org/project/keyd/versions>
